> ### ThreadLocal
>> * 原理 : 以`空间换时间的方式`为每一个线程都提供一份变量的副本，从而实现同时访问而互不干扰
>> * 侧重点 : 多线程中让每个线程之间的数据相互隔离
>> * 特点 : 
>>> * 传递数据 -- 在同一线程，不同组件中传递公共变量 （解耦）
>>> * 线程隔离 -- 在线程并发的情况下，各个线程的数据之间的数据相互隔离。（避免同步方式带来的性能损失）
>> * 与synchronized区别？ -- 并发效率的问题
